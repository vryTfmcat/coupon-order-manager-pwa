# 券食日历原型

在线体验：[coupon-order-manager-pwa.vercel.app](https://coupon-order-manager-pwa.vercel.app/)

![券食日历使用示例](./截屏2026-07-03%2004.26.28.png)

这是根据 `优惠券订单管理系统框架0.md` 做的第一版 Web/PWA 原型。

## 已做

- 顶部未安排卡片区。
- 周视图、日视图和月视图。
- 00:00-24:00 时间轴。
- 新增红包、团购券、固定活动、食物、快递五类卡片。
- 团购券支持填写地点，并在卡片和日历事项中优先显示。
- 固定活动可作为多次使用的未安排模板卡，例如疯狂星期四、周三活动日；安排到日历后模板仍保留。
- 手动编辑名称、来源、价格、有效期、可用时间段、想使用程度、标签和备注。
- 把未安排卡片拖到日历上。
- 选中卡片后点击日历格，也可以安排到对应时间。
- 月视图中选中卡片后点击某一天，会按卡片的可用开始时间安排到当天。
- 把已安排卡片拖到其他日期或时间。
- 可以复制卡片，用于多份相同优惠券或食物分别安排。
- 本地保存，刷新页面后不丢。
- JSON 导入导出。
- 基础可用性检查：临期等还能用的提醒标黄；已过期、晚于有效期/保质期、不在可用时段等不可用情况标红。
- 已使用的日历安排保留在日历中并显示为灰色。

## 本地运行

直接打开 `index.html` 可以使用基础功能。

如果要测试 PWA 安装和离线缓存，在这个目录运行：

```bash
python3 -m http.server 5173
```

然后打开：

```text
http://localhost:5173/
```

开发验证时可以打开 `http://localhost:5173/?reset=1` 重置本机演示数据。
